# CSS : Quais propriedades estão sendo usadas?
<background-image> Define uma imagem de fundo para o elemento.
<background-size> Controla o tamanho da imagem de fundo.
<background-position> Define a posição da imagem de fundo.
<background-repeat> ontrola se a imagem de fundo será repetida.
<color> Define a cor do texto dentro do elemento.
<align-items> Usado em elementos flexíveis para alinhar itens no eixo transversal (vertical, se o flex-direction for padrão)
<background-color> Define a cor de fundo de um elemento.
<width> Define a largura de um elemento.
<height> Define a altura de um elemento.
<position> Define o tipo de posicionamento de um elemento
<left> Define a posição à esquerda de um elemento, quando ele tem position: absolute;.
<right> Define a posição à direita de um elemento, quando ele tem position: absolute;.
<text-align> Define o alinhamento horizontal do texto dentro de um elemento.
<display-flex> Define o modelo de layout flexível para o elemento, permitindo o alinhamento e distribuição
<flex> Define a flexibilidade do item dentro de um contêiner flexível.
<gap> Define o espaçamento entre os itens dentro de um contêiner flexível.
<font-family> Define a família de fontes para o texto.
<font-weight> Define o peso da fonte (espessura do texto)
<justify-self> Controla o alinhamento do item em relação à linha dentro de um contêiner flexível ou grid.
<position:absolute> Coloca o elemento de forma absoluta em relação ao seu elemento pai posicionado ou ao documento.
<widtheheight>  Define as dimensões dos elementos.
